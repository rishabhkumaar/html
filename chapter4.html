Chapter 4 - Lists, Tables & Forms

Lists

Lists are used to display content that represents a group of related items.

Unordered List

Used to list unordered items:

<ul>
    <li>Home</li>
    <li>About</li>
</ul>

Tables

The <table> tag is used to define tables in HTML. It is used to format and display tabular data.

  • <tr> – defines a table row
  • <td> – defines table data (cell)
  • <th> – defines a table header (cell)
  • <caption> – adds a caption/title to the table
  • <thead> – wraps the header section of a table
  • <tbody> – wraps the body section of a table

Colspan Attribute

This attribute is used to create cells spanning multiple columns:

<th colspan="3">Harry</th>  → Spans 3 columns

HTML Forms

An HTML form is used to collect input from the user. The <form> tag is used for this purpose: 

<form>
    --Elements of the Form--
</form>

There are different form elements for different types of user input:

  • Input element: can be of types like text, checkbox, radio, button, submit, or file
  • Textarea element: defines a multi-line text input; cols and rows attributes control the size
  • Select element: defines a dropdown list
Note: You don't have to remember all the tags; with practice, you'll memorize them naturally.

Embedding Videos

The <video> tag is used to play videos in HTML:

<video src="video.mp4" controls>Your browser does not support the video tag.</video>

Attributes for Video

You can use the following attributes with the <video> tag:

  • width – Adjusts the width of the video (height adjusts automatically)
  • autoplay, loop, controls – Add behavior like autoplaying, looping, or showing controls

Practice 3: Travel Booking Form

<form>
    <label for="name">Full Name:</label>
    <input type="text" id="name" name="name" required>

    <label for="email">Email Address:</label>
    <input type="email" id="email" name="email" required>

    <label for="destination">Destination:</label>
    <select id="destination" name="destination">
        <option value="paris">Paris</option>
        <option value="tokyo">Tokyo</option>
        <option value="london">London</option>
        <option value="newyork">New York</option>
    </select>

    <label for="date">Travel Date:</label>
    <input type="date" id="date" name="date">

    <label for="requests">Special Requests:</label>

    <input type="submit" value="Book Now" style="margin-top: 1rem;">
</form>
