Chapter 0 - Introduction

HTML → HyperText Markup Language

HTML is the language of the web. It is used to create the structure of websites. We use HTML tags to define the layout and content of a page.

With a good understanding of these tags and how to use them together, we can build beautiful and functional websites!

Why Do We Need CSS and JavaScript?

HTML provides the bare structure of a page — like the skeleton of a house.

CSS (Cascading Style Sheets) is used to style that structure — changing colors, layouts, fonts, and more.

JavaScript brings the page to life. It allows us to add interactivity — like responding to clicks, showing or hiding elements, animations, and more.

A Simple Analogy

HTML → The car's chassis and frame — the basic structure.
CSS → The car's paint and design — how it looks and feels.
JavaScript → The engine and electronics — how it behaves and responds.

In this course, we’ll start by learning how to create solid page structures using HTML, and then we’ll style and power them using CSS and JavaScript.

Installing VS Code

You can use any text editor to write HTML. In this course, we’ll use Visual Studio Code because it's lightweight, open-source, and developed by Microsoft.

To install it, just go to Google, search for “VS Code,” and download it from the official website.

Note:

Even Notepad can be used to write HTML, but editors like VS Code make the process much easier and more efficient.

We start by creating a file named index.html. This is a special filename — it's the default page shown when someone visits your website’s root.

A Basic HTML Page (with Explanation)

<!DOCTYPE html>            → Specifies this is an HTML5 document
<html lang="en">         → Root of an HTML page
  <head>                   → Contains metadata about the page
    <meta charset="UTF-8">    → Character encoding
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> → Responsive design
    <title>Document</title> → Sets the page title (shown in the browser tab)
  </head>
  <body>                   → Main content visible in the browser
    <h1>This is a heading</h1> → A top-level heading
    <p>My paragraph</p>       → A paragraph of text
  </body>
</html>

A tag is like a container for either content or other HTML tags! HTML Document → Browser → Rendered Page

Important Notes

→ head & body tags are children of HTML tag.
→ HTML is the parent of head & body tags.
→ Most of the HTML elements have opening & closing tag with content in between opening & closing tags.
→ Some HTML tags have no content. These are called EMPTY ELEMENTS
→ We can either use .htm or .html extension
→ You can use "Inspect Element" or "View Page Source" option from chrome to look into a website's HTML Code!
HTML element = Start tag + Content + End tag

Comments in HTML

Comments in HTML are used to mark text that should not be rendered by the browser. They help document the source code and make it easier to understand. 

<!-- This is an HTML comment -->

Case Sensitivity

HTML is a case-insensitive language, which means tags like <h1> and <H1> are treated the same by the browser. 

<h1> and <H1> are the same!

Chapter 1 - Practice Set

  1. Inspect your favourite website and change something on the page which is displayed.

    You can right-click on a webpage and select “Inspect” or press Ctrl + Shift + I to open Developer Tools. In the Elements tab, you can change text, colors, or even remove elements.
    Note: These changes are only temporary and local — they disappear when you refresh the page.

  2. Go to your favourite website and try to view the page source. Write the exact lines of code. Does it clone the website? Why?

    Right-click and choose “View Page Source” or press Ctrl + U to see the HTML. You can copy this into your own .html file.

    However, it won’t fully clone the website because:

    • Many assets like images, stylesheets, and JavaScript files are hosted elsewhere.
    • Dynamic parts of the site (like content loaded via JavaScript) don’t appear in the raw HTML.
    • Some websites use frameworks that generate the page after it's loaded — that code won't be visible in the source.

  3. Write any HTML code inside a text file. Does it work if you write it using Notepad?

    Yes! HTML can be written in any text editor, including Notepad. Just make sure to save the file with a .html extension, for example: index.html.

    Once saved, you can double-click the file to open it in your browser.

Chapter 2 - Basic HTML Tags

We can add elements inside the body tag to define the page layout.

HTML Element

Everything from the starting to the ending tag.

<body> → Opening Tag
→Content←
</body> → Closing Tag

HTML Attribute

Used to add more information corresponding to an HTML tag.
Example: 

<a href="https://rizzhub.kr.netlify.app">Rishabh</a>

We can either use single or double quotes in attributes.

The Heading Tag

Heading tags are used to mark headings in HTML. From h1 to h6, we have tags for the most important to the least important heading.

<h1>"h1" Heading Tag</h1>
<h2>"h2" Heading Tag</h2>
<h3>"h3" Heading Tag</h3>
<h4>"h4" Heading Tag</h4>
<h5>"h5" Heading Tag</h5>
<h6>"h6" Heading Tag</h6>

Note: We should not use HTML headings to make text bold.

The Paragraph Tag

Paragraph tags are used to add paragraphs to an HTML page.

<p>This is a paragraph</p>

The Anchor Tag

The anchor tag is used to add links to content inside an HTML page.

<a href="https://google.com">Click Me</a>

The img Tag

The img tag is used to add images in an HTML page.

<img src="image.jpg" alt="Image description">

Bold, Italic and Underline Tags

We can use bold, italic, and underline tags to highlight text as follows:

<b>This is bold</b>
<i>This is italic</i>
<u>This is underline</u>

The br Tag

The br tag is used to create line breaks in an HTML document.

The big and small Tags

We can make the text a bit larger or smaller using big and small tags respectively.

<big>This is big</big>
<small>This is small</small>

Subscript and Superscript

We can add subscript and superscript in HTML as follows:

<sub>This is Subscript</sub>
<sup>This is Superscript</sup>
This is subscript & This is superscript

The pre Tag

HTML always ignores extra spaces and newlines. To display text exactly as written, we use the pre tag. 

<pre>
    This    is  written
    using pre
    tag
</pre>

Chapter 3 - Creating a Page Layout

When we use the right tag in the right place, it results in a better page layout, better indexing by search engines, and a better user experience. We use the following tags to get the job done:

Website Layout

<header>
<main>
<footer>

Inside the <main> Tag

<main> → The main opening tag
<section> → A page section
<article> → A self-contained content block
<aside> → Content aside from the main content (e.g., ads)
</main> → The main closing tag

Creating a page like this is not mandatory, but it creates a readable and structured layout. Also, it is useful for SEO.

Link Attributes

<a href="/contact">Contact Us</a> → Opens in the same tab

<a href="/contact" target="_blank">Contact Us</a> → Opens in a new tab

We can place any content inside an anchor tag (like images, headings, etc.).
If the page is inside a directory, make sure to link it correctly.
The same applies to the <img> tag as well.

We can add links to images like this:

<a href="/about">
  <img src="image.jpg" width="120" alt="About Image">
</a>

Height will be set automatically

The <div> Tag

The <div> tag is often used as a container for other elements.
It is a block-level element. Always takes full width

The <span> Tag

The <span> tag is an inline container.
Takes as much width as necessary

Chapter 4 - Lists, Tables & Forms

Lists

Lists are used to display content that represents a group of related items.

HTML provides ordered (<ol>) and unordered (<ul>) lists to organize content in bullet or numbered formats.

Unordered List

An unordered list displays items with bullet points. Each item is wrapped in a <li> tag.

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Tables

The <table> tag is used to define tables in HTML. It is used to format and display tabular data.

  • <tr> – defines a table row
  • <td> – defines table data (cell)
  • <th> – defines a table header (cell)
  • <caption> – adds a caption/title to the table
  • <thead> – wraps the header section of a table
  • <tbody> – wraps the body section of a table

Tables organize data into rows and columns using <table>, <tr>, <th>, and <td> tags. 

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>22</td>
  </tr>
</table>

Colspan Attribute

The colspan attribute merges columns in a table. It's added to a <td> or <th>.

<td colspan="2">Merged Cell</td>

HTML Forms

An HTML form is used to collect input from the user. The <form> tag is used for this purpose: 

<form>
    --Elements of the Form--
</form>

There are different form elements for different types of user input:

  • Input element: can be of types like text, checkbox, radio, button, submit, or file
  • Textarea element: defines a multi-line text input; cols and rows attributes control the size
  • Select element: defines a dropdown list
Note: You don't have to remember all the tags; with practice, you'll memorize them naturally.

Forms collect user input using elements like <input>, <textarea>, and <button>.

<form>
  <input type="text" placeholder="Name" />
  <input type="email" placeholder="Email" />
  <button>Submit</button>
</form>

Embedding Videos

Use the <video> tag to embed videos. You can specify controls, autoplay, and more. 

<video controls width="300">
  <source src="video.mp4" type="video/mp4">
</video>

Attributes for Video

You can use the following attributes with the <video> tag:

  • controls – Shows play, pause, and volume buttons
  • autoplay – Starts playing automatically
  • loop – Repeats the video
  • muted – Starts the video with sound off
  • poster – Sets an image to show before the video plays

Chapter 5 - HTML & SEO

SEO (Search Engine Optimization) is crucial for improving your website’s visibility on search engines. In this chapter, we explore how HTML can contribute to better on-page SEO practices.

Types of SEO

  • On-Page SEO: Optimizing content and HTML source code (keywords, meta tags, headings).
  • Off-Page SEO: Link building, social media sharing, external signals.
  • Technical SEO: Improving backend aspects like site speed, mobile-friendliness, and crawlability.

HTML Techniques for On-Page SEO

Some important techniques include:

  • Using proper heading structure with <h1> to <h6>.
  • Adding <meta name="description"> and <title> tags.
  • Using <alt> attributes for images.
  • Clean and semantic HTML using <header>, <nav>, <main>, <footer>, etc.
  • Adding internal links with descriptive anchor text.

Here are some practical steps HTML developers can take to improve on-page SEO:

  • Title Tag: Create a concise and descriptive <title> that includes relevant keywords.
  • Meta Description: Add a meta tag to summarize the page content for search engines and users.
<meta name="description" content="Learn essential HTML techniques for improving SEO on your website.">
  • URL Slug: Use clean, readable, and keyword-rich URLs (e.g., /seo-tips).
  • Meta Keywords (optional): Though outdated, still supported by some systems.
<meta name="keywords" content="HTML, SEO, web development">
  • Meta Author Tag: Specify the page author.
<meta name="author" content="Rishabh">
  • Favicon: Add a recognizable favicon for branding.
<link rel="icon" href="/favicon.ico" type="image/x-icon">
  • Optimize Media: Compress images and minify CSS/JS to improve page load speed.
  • Clean Code: Remove unused HTML, CSS, and JavaScript files.
  • Alt Text: Provide descriptive alt attributes for all images to improve accessibility and SEO.
