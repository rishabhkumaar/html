Chapter 2 - Basic HTML Tags

We can add elements inside the body tag to define the page layout.

HTML Element

Everything from the starting to the ending tag.

<body> → Opening Tag
→Content←
</body> → Closing Tag

HTML Attribute

Used to add more information corresponding to an HTML tag.
Example: 

<a href="https://rizzhub.kr.netlify.app">Rishabh</a>

We can either use single or double quotes in attributes.

The Heading Tag

Heading tags are used to mark headings in HTML. From h1 to h6, we have tags for the most important to the least important heading.

<h1>"h1" Heading Tag</h1>
<h2>"h2" Heading Tag</h2>
<h3>"h3" Heading Tag</h3>
<h4>"h4" Heading Tag</h4>
<h5>"h5" Heading Tag</h5>
<h6>"h6" Heading Tag</h6>

Note: We should not use HTML headings to make text bold.

The Paragraph Tag

Paragraph tags are used to add paragraphs to an HTML page.

<p>This is a paragraph</p>

The Anchor Tag

The anchor tag is used to add links to content inside an HTML page.

<a href="https://google.com">Click Me</a>

The img Tag

The img tag is used to add images in an HTML page.

<img src="image.jpg" alt="Image description">

Bold, Italic and Underline Tags

We can use bold, italic, and underline tags to highlight text as follows:

<b>This is bold</b>
<i>This is italic</i>
<u>This is underline</u>

The br Tag

The br tag is used to create line breaks in an HTML document.

The big and small Tags

We can make the text a bit larger or smaller using big and small tags respectively.

<big>This is big</big>
<small>This is small</small>

Subscript and Superscript

We can add subscript and superscript in HTML as follows:

<sub>This is Subscript</sub>
<sup>This is Superscript</sup>
This is subscript & This is superscript

The pre Tag

HTML always ignores extra spaces and newlines. To display text exactly as written, we use the pre tag.

<pre>
    This    is  written
    using pre
    tag
</pre>

Chapter 2 - Practice Set

  1. Create an HTML page with a heading (title heading), a primary heading and a subheading. Which tags did you use?

    For the page, I used the following HTML tags:

    • <title> for the browser title bar
    • <h1> for the primary heading
    • <h3> for the subheading
    Here’s an example: 

    <!DOCTYPE html>
<html>
<head>
  <title>Title Heading</title>
</head>
<body>
  <h1>This is the Main Heading</h1>
  <h3>This is the Subheading</h3>
</body>
</html>
  2. Create a page with 5 wallpaper images taken from the internet.

    You can use the <img> tag with direct image URLs from the internet. Example: 

    <img src="https://source.unsplash.com/random/300x200?wallpaper1" alt="Wallpaper 1">
<img src="https://source.unsplash.com/random/300x200?wallpaper2" alt="Wallpaper 2">
<img src="https://source.unsplash.com/random/300x200?wallpaper3" alt="Wallpaper 3">
<img src="https://source.unsplash.com/random/300x200?wallpaper4" alt="Wallpaper 4">
<img src="https://source.unsplash.com/random/300x200?wallpaper5" alt="Wallpaper 5">
  3. Use br and hr tags to display a piece of text with line breaks.

    You can use <br> for a new line and <hr> to insert a horizontal line. 

    This is line one.<br>
This is line two.<br>
<hr>
This is after the horizontal line.
  4. Try to write the following chemical equation using HTML: C + O2 → CO2

    Use the <sub> tag to write subscripts:

    C + O<sub>2</sub> → CO<sub>2</sub>

    Output: C + O2 → CO2

  5. Try to write a Wikipedia-like article using HTML.

    Here's a basic example of a Wikipedia-style article:

    <!DOCTYPE html>
<html>
<head>
  <title>Albert Einstein - Wikipedia</title>
</head>
<body>
  <h1>Albert Einstein</h1>
  <p>Albert Einstein was a German-born theoretical physicist who developed the theory of relativity, one of the two pillars of modern physics.</p>
  <h2>Early Life</h2>
  <p>Einstein was born in the Kingdom of Württemberg in the German Empire on 14 March 1879.</p>
  <h2>Contributions</h2>
  <p>Einstein is best known for his equation E = mc<sup>2</sup> and for winning the Nobel Prize in Physics in 1921.</p>
</body>
</html>
